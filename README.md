<h1>Construction Walk</h1>

<i>A 2D Unity project focused on fundamental physics mechanics, featuring realistic water buoyancy, player jumping/climbing systems, and interactive collectibles. Built to demonstrate core physics concepts through hands-on implementation, Construction Walk highlights practical rigidbody behavior, collision detection, and environmental interaction design.</i>
